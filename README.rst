Python.

- **Contact:** http://slack.simpeg.xyz
- **Zenodo:** https://doi.org/10.5281/zenodo.3229006


Workflow
========

To create the gallery:

.. code-block:: console

    make install
    conda activate emg3d-gallery
    make doc

To remove the environment:

.. code-block:: console

    make remove
